Supervision avec Gatus

> 1. PRÉSENTATION

Gatus est un outil de surveillance d'état (healthcheck) qui permet de vérifier la disponibilité des services via des requêtes HTTP, ICMP ou TCP, tout en offrant une page de statut claire et intuitive.

> 2. CONFIGURATION & ALERTING

Le projet repose sur une configuration déclarative simple et efficace :

  • Fichier YAML : Définition des endpoints à surveiller et des conditions de succès (temps de réponse, code HTTP 200).
  • Interface Web : Visualisation de l'historique de disponibilité (Uptime) sous forme de barres de santé.
  • Alerting : Intégration de Webhooks pour l'envoi immédiat de notifications sur Discord en cas de défaillance d'un service.

> 3. ANALYSE CRITIQUE
