Infrastructure LAMP & MariaDB

> 1. DESCRIPTION

Installation et configuration complète d'un serveur web sous Debian. Ce projet englobe la gestion du serveur HTTP Apache, l'interprèteur PHP et le système de gestion de base de données MariaDB.

> 2. CONFIGURATION & SÉCURITÉ

Le déploiement a été réalisé en suivant les bonnes pratiques d'administration système :

  • Installation Apache : Configuration des Virtual Hosts et gestion des droits sur /var/www/html.
  • Base de Données : Sécurisation de MariaDB (mysql_secure_installation) et création d'utilisateurs avec privilèges restreints.
  • Accès Distant : Configuration de l'accès à distance sécurisé pour l'administration de la base de données et du serveur.
