Sécurisation des accès avec Teleport

> 1. PROBLÉMATIQUE

Sécuriser et auditer les accès distants SSH vers une infrastructure de serveurs Debian sans exposer les ports directement sur Internet, tout en supprimant l'usage des clés SSH statiques.

> 2. MISE EN ŒUVRE DU BASTION

Le déploiement s'est déroulé sur Debian 12 avec les étapes clés suivantes :

  • Installation : Configuration du dépôt officiel et installation du service Teleport.
  • Enrôlement de Nœuds : Ajout de serveurs distants (nœuds) au cluster via des tokens d'invitation sécurisés.
  • Interface Web : Accès à une console unifiée permettant la gestion des ressources et l'ouverture de sessions SSH directement dans le navigateur.

> 3. FONCTIONNALITÉS AVANCÉES

La solution permet l'enregistrement vidéo des sessions (replay) et fournit un journal d'audit complet de chaque commande tapée par les administrateurs, garantissant une traçabilité totale exigée dans les environnements critiques comme le secteur hospitalier.
