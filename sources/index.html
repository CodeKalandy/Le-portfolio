RETOUR AU TERMINAL

SOURCES & CRÉDITS

Stack technique et ressources utilisées pour la réalisation de ce portfolio.

FRAMEWORKS & LIBS

  • Bootstrap 5.3 :
    Utilisé pour la structure responsive (Grid System) et les composants CSS de base (Boutons, Cartes).
  • AOS (Animate On Scroll) :
    Librairie JavaScript légère pour gérer les animations d'apparition des éléments lors du défilement.
  • Bootstrap Icons :
    Banque d'icônes vectorielles (SVG) pour les illustrations.

ASSISTANCE IA

Ce projet a été développé avec l'assistance d'Intelligences Artificielles Génératives (LLM) pour :

  • Génération de code : Structure HTML5 et scripts JS complexes (Moteur du jeu Terminus, Matrix Rain).
  • Débogage : Correction des erreurs de syntaxe CSS et logique JS.
  • Idéation : Brainstorming pour les fonctionnalités (Mode Furtif, Énigmes).
Note : L'intégration, l'adaptation et la validation finale du code ont été réalisées par mes soins.

API & SERVICES

  • Ipify API :
    Service externe (REST API) utilisé pour récupérer et afficher l'adresse IP publique du visiteur dans le widget de monitoring.
  • Google Fonts :
    Polices d'écriture utilisées : 'Orbitron' (Titres), 'Share Tech Mono' (Terminal), 'Roboto' (Corps).

DÉVELOPPEMENT

  • HTML5 / CSS3 :
    Design personnalisé "Glassmorphism" et "Cyberpunk". Gestion des animations CSS (Keyframes) pour le scan et les effets néon.
  • JavaScript (Vanilla) :
    Programmation de la logique du Terminal, du jeu Terminus (système de fichiers virtuel) et de la calculatrice IP sans framework JS lourd (type React/Vue).

Portfolio développé dans un but pédagogique pour le BTS SIO SISR.
