TERMINUS_WIKI_V1.0

DOCUMENT CLASSIFIÉ "TOP SECRET". CLIQUEZ SUR LES ZONES NOIRES POUR DÉCHIFFRER LES SOLUTIONS.

LVL 01 L'Exploration

Objectif : Lire le contenu du fichier texte présent dans le dossier.

SPOILER
ls cat readme.txt
La commande 'ls' liste les fichiers, 'cat' affiche le contenu.
LVL 02 Fichier Caché

Objectif : Trouver un fichier qui commence par un point (ex: .config).

SPOILER
ls -a cat .secret
L'option '-a' (all) permet de voir les fichiers cachés sous Linux.
LVL 03 Navigation Relative

Objectif : Se déplacer dans un autre dossier du système.

SPOILER
cd /var/log cat syslog
Vous pouvez aussi faire 'cd ..' pour remonter d'un cran.
LVL 04 Permission Root

Objectif : Lire un fichier protégé réservé à l'administrateur.

SPOILER
sudo su cat shadow
'sudo su' vous fait passer en utilisateur 'root' (SuperUser).
LVL 05 Nettoyage

Objectif : Supprimer un fichier dangereux.

SPOILER
cd /tmp rm virus.exe
'rm' signifie Remove. Attention, c'est définitif !
LVL 06 Création Dossier

Objectif : Créer un nouveau répertoire pour les sauvegardes.

SPOILER
cd /home/guest mkdir backup
'mkdir' = Make Directory.
LVL 07 Info Réseau

Objectif : Connaître son adresse IP.

SPOILER
ip a
L'ancienne commande 'ifconfig' fonctionne aussi.
LVL 08 Recherche (Grep)

Objectif : Trouver une ligne spécifique dans un gros fichier.

SPOILER
cat users.txt | grep admin
Le symbole '|' (pipe) envoie le résultat de 'cat' vers la commande 'grep' qui filtre.
LVL 09 Création Fichier

Objectif : Créer un fichier vide rapidement.

SPOILER
touch test.log
'touch' met à jour l'heure d'un fichier ou le crée s'il n'existe pas.
LVL 10 LE BOSS FINAL

Objectif : Combinez vos compétences (Root + Navigation + Suppression).

SPOILER
sudo su cd /root rm core.sys
Félicitations, vous avez détruit le cœur du système (virtuellement) !
